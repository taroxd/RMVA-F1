RPG::Animation::Frame

动画帧的数据类。

超类

  • Object

调用者

  • RPG::Animation

属性

cell_max

图象元数。相当于本帧中图象元的最大编号。

cell_data

包含图象元内容的二维数组（Table ）。

使用 cell_data[cell_index, data_index] 的形式。

data_index 的范围是从 0 到 7 ，分别代表下面的各种图象元信息：

0：图象元号
1：X 坐标
2：Y 坐标
3：放大率
4：旋转角度
5：横向翻转
6：不透明度
7：合成方式

图象元号实际上为RPGVX编辑器中所显现的图象元编号，编号为-1则代表该图象元不被使用。

定义

class RPG::Animation::Frame
  def initialize
    @cell_max = 0
    @cell_data = Table.new(0, 0)
  end
  attr_accessor :cell_max
  attr_accessor :cell_data
end
