RPG::System

系统的数据类。

超类

  • Object

属性

game_title

标题文字。

version_id

检查更新用的随机数，RPG Maker VX每次存档都会变更此数值。

japanese

在日文版本中始终开启的开关。

party_members

初始队伍。角色 ID 编号的数组。

currency_unit

货币单位。

window_tone

窗口颜色（Tone ）。

elements

属性列表。以属性 ID 编号为索引的字符串数组，0 号元素为 nil。

skill_types

技能类型列表。以技能类型 ID 编号为索引的字符串数组，0 号元素为 nil。

weapon_types

武器类型列表。以武器类型 ID 编号为索引的字符串数组，0 号元素为 nil。

armor_types

防具类型列表。以防具类型 ID 编号为索引的字符串数组，0 号元素为 nil。

switches

开关列表。以开关 ID 编号为索引的字符串数组，0 号元素为 nil。

variables

变量列表。以变量 ID 编号为索引的字符串数组，0 号元素为 nil。

boat

小船设置（RPG::System::Vehicle）。

ship

大船设置（RPG::System::Vehicle）。

airship

飞艇设置（RPG::System::Vehicle）。

title1_name

标题背景画面的文件名。

title2_name

标题前端画面的文件名。

opt_draw_title

绘制标题文字的开关。

opt_use_midi

启动时初始化 MIDI 的开关。

opt_transparent

初始时玩家行走图透明的开关。

opt_followers

队列行进（开火车）的开关。

opt_slip_death

慢性伤害可导致无法战斗的开关。

opt_floor_death

地形伤害可导致无法战斗的开关。

opt_display_tp

在战斗画面显示特技值的开关。

opt_extra_exp

替补队员可获得经验值的开关。

title_bgm

标题画面背景音乐（RPG::BGM）。

battle_bgm

战斗画面背景音乐（RPG::BGM）。

battle_end_me

战斗结束音效（RPG::ME）

gameover_me

游戏结束音效（RPG::ME）。

sounds

声效设置（RPG::SE 数组）。

start_map_id

玩家初始位置的地图编号。

start_x

玩家初始位置的 X 坐标。

start_y

玩家初始位置的 Y 坐标。

terms

用语（RPG::System::Terms）。

test_battlers

战斗测试设置（RPG::System::TestBattler）。

test_troop_id

战斗测试中的敌群编号。

battleback1_name

战斗测试背景下半部文件名。

battleback2_name

战斗测试背景上半部文件名。

battler_name

编辑器内部使用，战斗图文件名。

battler_hue

编辑器内部使用，战斗图档色调调整值（0～360）。

edit_map_id

编辑器内部使用，目前正在编辑的地图 ID 编号。

内部类

  • RPG::System::Vehicle
  • RPG::System::Terms
  • RPG::System::TestBattler

class RPG::System
  def initialize
    @game_title = ''
    @version_id = 0
    @japanese = true
    @party_members = [1]
    @currency_unit = ''
    @elements = [nil, '']
    @skill_types = [nil, '']
    @weapon_types = [nil, '']
    @armor_types = [nil, '']
    @switches = [nil, '']
    @variables = [nil, '']
    @boat = RPG::System::Vehicle.new
    @ship = RPG::System::Vehicle.new
    @airship = RPG::System::Vehicle.new
    @title1_name = ''
    @title2_name = ''
    @opt_draw_title = true
    @opt_use_midi = false
    @opt_transparent = false
    @opt_followers = true
    @opt_slip_death = false
    @opt_floor_death = false
    @opt_display_tp = true
    @opt_extra_exp = false
    @window_tone = Tone.new(0,0,0)
    @title_bgm = RPG::BGM.new
    @battle_bgm = RPG::BGM.new
    @battle_end_me = RPG::ME.new
    @gameover_me = RPG::ME.new
    @sounds = Array.new(24) { RPG::SE.new }
    @test_battlers = []
    @test_troop_id = 1
    @start_map_id = 1
    @start_x = 0
    @start_y = 0
    @terms = RPG::System::Terms.new
    @battleback1_name = ''
    @battleback2_name = ''
    @battler_name = ''
    @battler_hue = 0
    @edit_map_id = 1
  end

  attr_accessor :opt_transparent
  attr_accessor :opt_followers
  attr_accessor :opt_slip_death
  attr_accessor :opt_floor_death
  attr_accessor :opt_display_tp
  attr_accessor :opt_extra_exp
  attr_accessor :window_tone
  attr_accessor :title_bgm
  attr_accessor :battle_bgm
  attr_accessor :battle_end_me
